커리큘럼

5일, 총 30시간

  1. Day1파이썬 기초와 환경 설정 2024.01.22(월) | 10:00 ~ 17:00

    학습 목표

    • 파이썬에 기본 개념 이해하기
    • colab 사용법
    • 변수와 타입 이해하기
    • 시퀀스 자료형에 이해와 활용
    • 입력과 출력 이해와 활용

    주요 내용

    • 파이썬과 실행 환경 소개
    • 파이썬이 실행되는 프로세스 이해
    • colab 사용법과 단축키
    • 변수의 타입(정수부터 None까지)과 메서드
    • 연산(산술, 비교, 논리, 할당, is, in 등)
    • 파이썬 버전별 입출력 역사와 f-string 문법

  2. Day2컬렉션 자료형과 함수 2024.01.23(화) | 10:00 ~ 17:00

    학습 목표

    • 컬렉션 자료형에 이해와 활용
    • 함수에 대한 이해와 활용

    주요 내용

    • 리스트, 튜플, 딕셔너리, 셋의 메서드 살펴보기
    • 함수 만들어보기
    • 아규먼트와 파라미터
    • 지역 변수와 전역 변수
    • built-in function 살펴보기

    일급함수와 고차함수, 재귀함수, 클로저, 데코레이터는 4일차에 편성되어 있습니다.

  3. Day3 조건문과 반복문 2024.01.24(수) | 10:00 ~ 17:00

    학습 목표

    • 조건문의 이해와 활용
    • 반복문의 이해와 활용

    주요 내용

    • if, elif, else를 통한 조건 분기 만들기
    • match를 통한 조건 분기 만들기
    • for, while, for + else, while + else를 통한 반복문 만들기
    • 중첩 조건문과 중첩 반복문 이해하기
    • break, continue, pass 차이점 이해와 활용하기

  4. Day4클래스와 Python 심화 과정 2024.01.25(목) | 10:00 ~ 17:00

    학습 목표

    • 클래스의 이해와 활용
    • 매직메서드 이해하고 활용하기
    • 실무에서 자주 사용하는 고급 문법 살펴보기
    • 실무에서 사용하는 테스팅과 디버깅 살펴보기
    • 모듈의 이해와 활용
    • 예외처리의 이해와 활용

    주요 내용

    • 클래스를 정의하고 적절한 메직메서드를 통하여 실무에서 사용하는 다양한 클래스 살펴보기
    • 실무에서 사용하는 클래스들의 형태 살펴보기
    • 메서드 체이닝, 일급함수와 고차함수, 클로저, 데코레이터, 람다, args, kwargs, 이터레이터와 제너레이터, nonlocal, 코루틴 등 실무에서 자주 사용하는 고급 문법 살펴보기
    • 디버깅 툴과 테스팅 모듈을 활용하여 실무에서 어떻게 테스트하는지 확인하고 코드 살펴보기
    • 모듈의 작동 원리와 읽는 순서 확인하기
    • 커스텀 예외 만들기와 자주 나오는 애러 정리하고 실무에서 사용하는 커스텀 애러 살펴보기

  5. Day5코딩테스트를 통한 이해 점검하기 2024.01.26(금) | 10:00 ~ 17:00

    학습 목표

    • 코딩테스트 출제 경향과 유형 분석
    • 유형 살펴보기
    • 코딩테스트 전략 수립하기
    • 정규표현식 이해와 활용

    주요 내용

    • 문제 출제 경향 분석
    • 기업별 코딩테스트 유형 분석
    • 코딩테스트 전략 수립하기
    • 성장 전략 수립하기
    • Lv0로 실력 점검하고 부족한 문법이 있는지 확인
    • 코딩테스트 맛보기
    • 정규표현식 사용 방법과 문제 활용 방법

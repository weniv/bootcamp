프론트엔드 개발자란?

웹/앱 서비스에서사용자가 눈으로 보는 영역을
구축하고, 기능을 구현하는 개발자입니다

프론트엔드 개발자가 되기 위해서는

다음 세 가지의 핵심 역량이 반드시 필요합니다

  • HTML, CSS, JavaScript와 같은 웹 기본 언어

  • 디자이너와 협업하여 디자인을 실제 화면으로 구현할 수 있는 역량

  • 모든 사용자가 불편 없이 서비스를 이용할 수 있도록 하는 웹 접근성에 대한 이해

5개월간의 몰입 교육을 통해
프론트엔드 개발자 핵심 역량을 탄탄히 쌓아보세요

이스트소프트 X 위니브

프론트엔드 개발자 양성 과정 오르미 로드맵
  1. 1달 차개발 기초와 협업 툴 익히기
    • 오리엔테이션 & GitHub 기초

      학습 방법을 배우고, GitHub로 코드를 관리하는 기본 방법을 익힙니다.

    • HTML/CSS & Figma

      HTML과 CSS의 기초를 배우고, 웹 표준과 웹 접근성을 준수하는 코드를 작성합니다. 또한 Figma로 디자인 파일을 구현하는 법을 배우며 실무 프로젝트 기반으로 학습합니다.

  2. 2달 차프로젝트와 JavaScript
    • HTML/CSS 프로젝트

      Figma 디자인을 바탕으로 HTML/CSS 프로젝트를 완성하고 코드 리뷰를 통해 더 나은 코드를 작성하는 방법을 배웁니다.

    • JavaScript 기초

      JavaScript의 기본 문법부터 비동기 프로그래밍, DOM 조작까지 배우며 실무에 필요한 스킬을 쌓습니다. 학습용 API를 통해 실제 데이터를 다루는 방법도 익힙니다.

  3. 3달 차네트워크와 심화 학습
    • 네트워크 & jQuery & Sass

      웹에서 데이터를 주고받는 방법(REST API, HTTP/HTTPS 등)과 jQuery, Sass 등의 도구를 학습합니다.

    • 알고리즘 & 이력서

      코딩 테스트를 준비하며 기본 알고리즘을 배우고, 이력서를 작성하는 방법을 익혀 취업 준비를 돕습니다.

  4. 4달 차React와 TDD
    • React 기초 및 심화

      React의 기본부터 Hooks, 상태 관리, 비동기 통신까지 학습합니다.

    • 테스트 주도 개발(TDD) & Webpack

      TDD로 코드 품질을 높이는 방법을 배우고, Webpack으로 프로젝트를 빌드 및 관리하는 기술을 익힙니다.

  5. 5달 차TypeScript와 최종 프로젝트
    • TypeScript & Firebase

      TypeScript로 코드의 안정성을 높이고, Firebase를 통해 백엔드를 쉽게 구축하는 방법을 배웁니다.

    • GitHub 심화 & 최종 프로젝트

      실무에서 사용하는 GitHub의 다양한 기능을 배우고, 실무형 프로젝트를 진행하여 프론트엔드 개발자로서의 역량을 완성합니다.

    대한민국의 모든 교육자가 위니브 강사님들만 같았으면 참 좋겠다 싶을 정도였다.

  1. 1주차오리엔테이션, GitHub 기초

    강의 내용

    공부하는 법을 배우고, GitHub를 통해 코드를 관리하는 방법을 간단하게 배웁니다. 앞으로 어떻게 성장할 것인지, 우리가 어떤 식으로 공부해야 하는지에 대한 방향을 제시합니다.

  2. 2주차 ~ 4주차HTML/CSS와 figma

    강의 내용

    HTML/CSS의 문법을 견고하게 익혀 웹 표준과 웹 접근성을 준수하는 시멘틱한 마크업과 모듈화된 CSS 설계 방법을 학습합니다. 이 교육은 위니브의 위니북스* 프로젝트 등 실무에서 사용했던 수십개의 프로젝트의 사례를 기반합니다.

    또한 피그마의 기본 사용법을 익혀 기업의 입사문제 기출문제 수준의 디자인을 바탕으로 구현하는 능력을 키우고, 실무에서 요구하는 디자인 구현 능력과 문제 해결 능력을 키웁니다.

    * 위니북스 링크: https://books.weniv.co.kr

  3. 5주차HTML/CSS 프로젝트

    강의 내용

    공부했던 HTML/CSS를 바탕으로 프로젝트를 진행합니다. 프로젝트는 실무에서 하는 것처럼 피그마 디자인이 준비되어 있습니다. 이 디자인을 바탕으로 구현하고, 구현한 코드를 상세히 리뷰합니다. `div` 태그는 왜 최후의 수단으로 사용하는지, `header`에 이미지에 왜 접근성 텍스트를 넣어야 하는지 등을 코드리뷰를 통해 배웁니다.

  4. 6주차 ~ 8주차JavaScript

    강의 내용

    최신 ECMAScript 문법을 통해 JavaScript의 기본문법부터 비동기 프로그래밍, DOM 조작, 이벤트 위임, 클로저, 프로토타입, 클래스, 모듈 등을 학습합니다. 이 과정을 통해 면접에서 나올 여러 JavaScript 질문을 대비하고, 기본기를 다지며, 앞으로 어떠한 방향으로 기술을 쌓아야 할지 방향을 제시합니다. 대부분의 기술은 실제 실무에서 사용하는 것처럼 준비되어 있습니다. 예를 들어, 비동기 통신의 경우 학습용 API인 위니브 API가 준비되어 있습니다.

  5. 9주차네트워크, jQuery, Sass, 알고리즘, 이력서

    강의 내용

    프론트엔드 개발자에게 필요한 네트워크 지식을 학습합니다. 서로 주고 받는 데이터, REST API, HTTP, HTTPS, CORS 등 필수적인 네트워크 지식을 학습합니다.

    또한 요즘에는 많이 사용하지 않지만, 레거시 코드 수정을 위해 jQuery를 가볍게 학습하고 jQuery가 어떻게 만들어진 것인지, 앞으로 어떤 라이브러리나 프레임웤이 유행할지 등을 함께 알아봅니다.

    Sass를 통해 CSS를 더욱 효율적으로 작성하는 방법을 학습합니다.

    코딩테스트를 위한 알고리즘을 학습합니다. 위니브에는 베이스 문제 100제, 베스트 20 문제 등 다양한 코딩테스트 문제를 풀 수 있는 서비스가 준비되어 있습니다. 이를 통해 코딩테스트를 준비하고, 어떤 회사에서 코딩테스트가 어떠한 유형으로 가장 많이 나오는지 분석한 다음, 그에 맞게 대비할 수 있습니다.

    이력서 작성법을 학습합니다. 위니브가 출판한 '개발자 이력서 작성가이드' 책을 기반으로 개발자 이력서를 어떻게 작성해야 눈에 띄게 할 수 있는지, 어떤 내용을 넣어야 하는지, 어떤 내용을 넣으면 안 되는지 등을 학습합니다. 이력서 첨삭을 통해 자신의 이력서를 개선하고, 알맞은 연봉 수준을 책정하고, 취업에 대한 방향을 제시합니다.

  6. 10주차프로젝트

    강의 내용

    앞서 배운 내용을 기반으로 이미 피그마로 디자인 되어 있는 프로젝트를 구현합니다. 이 프로젝트부터 포트폴리오로 가져갈 수 있도록 주제나 이미지 등 콘텐츠 수정이 가능합니다. 이 프로젝트를 통해 JavaScript를 제대로 사용하는지, 적절한 코드를 사용하였는지 확인하고, 단체로 코드리뷰를 받으며 서로 코드를 어떻게 구현하였는지 공유하고 비교합니다.

  7. 11주차 ~ 14주차React/TDD/Webpack

    강의 내용

    React의 핵심 원리부터 심화까지 학습합니다. React의 기본적인 사용법부터, React Hooks, Context API, React Router 등을 폭넓게 학습합니다. 이 과정을 통해 React를 통해 어떻게 프로젝트를 구성하고, 어떻게 상태를 관리하는지, 어떻게 비동기 통신을 하는지 등을 학습합니다. 리엑트의 생명주기를 이해하고, 상태 관리 라이브러리인 리코일(Recoil)과 리덕스(Redux)를 함께 학습합니다.

    React를 통해 학습을 진행하며 테스트 주도 개발(TDD)을 어떻게 적용할 수 있는지 학습합니다. TDD를 통해 코드의 품질을 높이고, 유지보수성을 높이는 방법을 학습합니다. 또한 Webpack을 통해 프로젝트를 어떻게 빌드하고, 어떻게 모듈을 관리하는지 학습합니다.

  8. 15주차 ~ 18주차TypeScript, firebase

    강의 내용

    코드의 가독성과 유지보수성을 높이는 TypeScript의 기본문법과 실무에서 자주 사용하는 고급 기능을 학습합니다. TypeScript를 React와 함께 사용하는 방법을 배우고, 타입 안정성을 갖춘 React 컴포넌트와 상태 관리 코드를 작성하는 기술을 익힙니다.

    프론트엔드 개발자가 익힐 수 있는 Firebase의 간단한 기능을 활용하여 빠르고 쉽게 백엔드를 구축할 수 있는 방법을 익힙니다. Firebase를 통해 프론트엔드 개발자가 백엔드에 대한 깊은 이해 없이 개발할 수 있는 데이터 베이스 관리, 인증 및 권한 관리 시스템을 구현하는 기술을 습득합니다.

  9. 19주차 ~ 20주차GitHub 심화, 프로젝트

    강의 내용

    아래에 해당하시는 분들께서는 내일배움카드 발급 대상자가 아닙니다.

    - 현직 공무원

    - 사립학교 교직원

    - 졸업예정자 이외 재학생 (졸업예정자의 경우 발급 가능)

    - 연 매출 1억 5천만원 이상의 자영업자

    - 월 임금 300만원 이상인 대규모 기업 종사자(45세 미만)

    - 특수형태근로종사자

    ※ 내일배움카드 발급 가능 여부는 고용노동부 1350 또는 거주지 관할 고용센터에서 확인해 주세요.

  2. Q2.[지원 및 선발] 내일배움카드가 없는데 지원할 수 있나요?

    고용노동부 1350을 통해 카드 발급 가능 여부를 확인한 후, 빠르게 발급 신청해 주세요. 카드 발급에는 1주일 이상의 시간이 소요될 수 있으니, 가급적 은행 방문 수령을 추천드립니다. 실물 카드 수령이 완료되어야 수강 신청이 최종 확정됩니다.

  3. Q3.[지원 및 선발] 다른 기관에서 교육을 받았었는데 신청 가능할까요?

    이전에 K-디지털 트레이닝 수강 이력이 있다면 신청이 불가능합니다. 다만, K-디지털 트레이닝이 아닌 다른 과정을 수강하셨다면 문제없이 신청 가능합니다. HRD-Net에 로그인 후 '나의 훈련 이력' 페이지에서 이전 교육과정을 확인해 주세요.

  4. Q4.[지원 및 선발] 학교, 직장 등과 병행이 가능한가요?

    저희 교육 과정은 매일 오전 9시부터 오후 6시까지 풀타임 강의로 진행됩니다. 매일 전일제 수업에 온전히 참여 가능하신 분들을 우선 선발하고 있습니다.

  5. Q5.[교육] 비전공자인데 강의 듣는데 힘들지 않을까요?

    사전 설문을 통해 지원자분들의 사전 지식을 파악하고, 이를 바탕으로 커리큘럼에 반영하고 있습니다. 기존에 많은 비전공자분들이 수강했던 경험을 바탕으로, 강사진과 운영진이 세심하게 관리하여 걱정 없이 학습하실 수 있도록 도와드립니다.

  6. Q6.[교육] 교육 전에 미리 공부하면 좋은 것이 있을까요?

    최종 선발된 교육생분들께는 온라인 VOD 콘텐츠(STEP)를 제공해 드립니다. 해당 콘텐츠를 학습하면 수업 참여에 도움이 됩니다. 또는 기초 프로그래밍이나 AI 개념에 대한 기본적인 이해를 하고 오시면 더 좋습니다.

  7. Q7.[교육] 개인 일정 때문에 수업에 빠져야 하는 경우 어떻게 해야 하나요?

    병원 진료, 면접, 예비군 훈련 등의 사유로 수업에 불참해야 하는 경우, 관련 서류를 제출해 주시면 출결 처리를 도와드립니다. 수업 녹화본은 제공되지 않지만, 수업 자료와 교안을 드리고, 강사진과 멘토진이 수업 진도를 따라갈 수 있도록 지원해 드립니다.

  8. Q8.[교육] 팀 프로젝트는 온라인으로 진행하나요?

    강의뿐만 아니라 팀 프로젝트도 온라인으로 진행됩니다. Zoom과 Discord 등을 통해 비대면 협업을 지원해 드리며, 원활하게 프로젝트를 진행할 수 있도록 돕고 있습니다.
